# 言語ポリシー（日本語のみ）

このリポジトリの **サイト表示（HTML/JS）・Slack投稿・ボタン/リンク文言** は、原則 **日本語のみ** とします。

## ルール
- UIに英単語を入れない（例: `Top` / `My` / `List` / `Review` などは禁止）
- 外部リンクは **日本語で閲覧できるURL** を優先する
  - 株探（例: `kabutan.jp`）
  - TDnet公式（例: `www.release.tdnet.info`）
- 適時開示PDFは **TDnet公式（日本語）** をデフォルトにし、株探ミラーは閲覧不能時の予備として併記する
- 英語ソースが必要な場合でも、**表示名（ラベル）は日本語** にする

## 例外（必要最小限）
- 銘柄コード（例: 9983）、指数名の略号、企業の固有名詞
- PDF/一次資料が英語しかない場合は、UI上で日本語の要点（要約/翻訳）を併記する
